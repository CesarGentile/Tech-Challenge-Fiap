Este projeto tem como objetivo analisar o comportamento da população durante a pandemia de COVID-19 e identificar indicadores importantes para o planejamento hospitalar em caso de novos surtos, utilizando dados da pesquisa PNAD-COVID-19 do IBGE.

O processo envolve a organização da base em nuvem no Google BigQuery, seleção de até 20 variáveis relevantes (sintomas clínicos, características populacionais e impactos econômicos), execução de consultas SQL e desenvolvimento de análises em Python no Jupyter Notebook, com geração de visualizações e insights estratégicos.

As tecnologias utilizadas incluem Google Cloud Platform (BigQuery), SQL, Python, pandas e matplotlib.

Para executar o projeto, recomenda-se configurar o ambiente com as dependências necessárias e utilizar os notebooks ou scripts disponíveis para reproduzir as consultas e análises.

Os resultados obtidos permitem mapear grupos vulneráveis, entender a evolução temporal dos sintomas e internações, além de evidenciar impactos socioeconômicos relevantes, fornecendo recomendações acionáveis para gestores hospitalares em futuras crises de saúde.

O projeto demonstra a importância da modelagem analítica e da visualização de dados como suporte essencial para tomada de decisão em emergências sanitárias, com potencial de expansão para análises preditivas e novas abordagens de monitoramento epidemiológico.
